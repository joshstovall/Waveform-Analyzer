# Waveform-Analyzer

 Build & Analyze Sound Waves


> A simple application to help demonstrate fundamental properties of sound.


Features :

- 4 oscillators

- Spectral Analysis

- Oscilloscope

- FFT (Fast Fourier Transform) Analysis

- Frequency adjustments down to .0001

- Type in number to set frequency


Available Waveforms :

- Sine Wave

- Triangle Wave

- Saw Wave

- Square Wave
